The Pantry

This website allows users to add food items in their possession onto a virtual database and then search for recipes based off of those item and create a shopping-list for items not in their store. Users may also take advantage of a manual ingredient-based recipe search and an ingredient-substitute search feature. It also provides the user with the locations of grocers in their current area.

Space Gifs

A basic website which taps into the GIPHY API and allows users to search for space-themed gifs using the pre-created buttons. One may also utilize the text-box to add their own button with any desired topic.

Trivial Trivia

A timed quiz in which the user must answer a multiple-choice trivia question (subject related to dogs in this particular program) before the timer runs out.

Word Guess

A simple game of word-guess. The program chooses a country from an array of countries, and the user has to guess which country it is in 2x the amount in letters.

Crystal Collector

A simple guessing game where the user must reach the randomly-generated 'Number to Hit' without going over. There are four crystals, each with their own random values, for the user to click on and increase their current score number.
